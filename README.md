
   Create `server/.env`:
   ```env

   GOOGLE_CALLBACK_URL=http://localhost:5000/api/auth/google/callback

   CORS_ORIGINS=http://localhost:8080,http://localhost:5173
   PORT=5000
   NODE_ENV=development

   ```

### 3. Start Development Servers
